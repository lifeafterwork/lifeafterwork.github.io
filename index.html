CSS: Display and Positioning

CSS: Display and Positioning Cheatsheet

Declaration Description
z-index Specifies how far back or forward an element will appear on a webapage when it overlaps other elements.
position: fixed Pins element to a specific spot on a page.
position: relative Enables an element to be positioned relative to where it would have originally been on a web page.
position: absolute Enables an element to ignore sibling elements and instead be positioned relative to its cosest parent element that is positioned with "relative" or "absolute". The absolute value removed an element entirely from the document flow. By using the positioning attributes top, left, bottom and right, an element can be positioned anywhere as expected.
float determines hwo far left or how far right an element should float within its parent element.
clear specifies how an element should behave when it bumps into another element within the same containing element. Clear is usually used in combination with elemnts having the CSS float property. This detemines on which sides floating elements are allowed to float.
